# Data Science Capstone

Projeto final de **Data Science** com foco na análise e previsão de sucesso de pouso do primeiro estágio do foguete **Falcon 9 da SpaceX**. O objetivo é aplicar, de forma prática, etapas fundamentais de um pipeline de ciência de dados, desde a coleta e preparação dos dados até visualização, machine learning e dashboard interativo.

## Objetivo do projeto

A proposta deste projeto é analisar dados de lançamentos da SpaceX e construir modelos capazes de prever se o primeiro estágio do foguete conseguirá pousar com sucesso. Esse tipo de previsão é importante porque a reutilização do estágio reduz significativamente o custo de lançamentos. :contentReference[oaicite:1]{index=1}

## Etapas desenvolvidas

O projeto foi organizado em notebooks e scripts que cobrem diferentes fases do processo de análise:

- **Coleta de dados via API**
- **Web Scraping**
- **Limpeza e tratamento dos dados**
- **Análise exploratória com SQL**
- **Visualização de dados**
- **Mapas interativos com Folium**
- **Modelagem preditiva com Machine Learning**
- **Dashboard interativo com Dash e Plotly** :contentReference[oaicite:2]{index=2}
